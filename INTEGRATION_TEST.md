# 联调测试指南

## 1. 启动后端服务

```bash
cd ai-bi-product/backend
npm install
npm run dev
```

后端服务将在 http://localhost:3000 启动

## 2. 启动前端服务

```bash
cd ai-bi-product/frontend
npm install
npm run dev
```

前端服务将在 http://localhost:5173 启动

## 3. 测试用例

### 3.1 简单查询测试

| 测试查询 | 预期结果 |
|----------|----------|
| 销售额是多少 | 返回总销售额 |
| 有多少订单 | 返回订单数量 |
| 平均订单金额 | 返回平均值 |

### 3.2 GROUP BY 测试

| 测试查询 | 预期结果 |
|----------|----------|
| 按客户类型统计销售额 | 分组数据 + 图表 |
| 各产品类别的销售额对比 | 分组数据 + 图表 |
| 客户数量按城市分组 | 分组数据 + 图表 |

### 3.3 多条件测试

| 测试查询 | 预期结果 |
|----------|----------|
| 零售客户的销售额 | 带筛选的结果 |
| 已完成订单的销售额 | 带筛选的结果 |
| 最近7天的销售额 | 时间筛选结果 |

## 4. API 端点测试

### 4.1 Agent 查询

```bash
curl -X POST http://localhost:3000/api/agent/query \
  -H "Content-Type: application/json" \
  -d '{"query": "销售额是多少"}'
```

### 4.2 Agent 列表

```bash
curl http://localhost:3000/api/agent/list
```
