# 功能拆解清单（MVP版本）

## 一、模块一：极简数据接入模块

### 1.1 数据源管理

| 任务ID | 功能点 | 描述 | 优先级 | 技术依赖 |
|--------|--------|------|--------|----------|
| F01-01 | 数据源类型选择 | 支持8类数据源：MySQL、PostgreSQL、SQL Server、ClickHouse、Excel/CSV、金蝶/用友API、飞书/钉钉表单、CRM API | P0 | - |
| F01-02 | 连接表单 | 极简配置表单，仅必填项（名称、地址、端口、账号、密码、库名） | P0 | - |
| F01-03 | 测试连接 | 填完连接信息后测试连通性 | P0 | 对应数据库驱动 |
| F01-04 | 连接失败提示 | 返回人话提示（账号错误/网络不通等），不暴露底层报错 | P0 | - |
| F01-05 | 数据源列表 | 显示已接入的数据源，支持编辑/启用禁用/删除 | P0 | - |
| F01-06 | 删除数据源 | 删除时校验是否有保存的查询结果 | P1 | F03-02 |

### 1.2 AI元数据识别（后台自动执行）

| 任务ID | 功能点 | 描述 | 优先级 | 技术依赖 |
|--------|--------|------|--------|----------|
| F01-07 | 表结构扫描 | 连接成功后自动扫描全库表结构 | P0 | 数据源连接 |
| F01-08 | 字段类型识别 | 自动识别字段类型（数值/日期/字符串等） | P0 | F01-07 |
| F01-09 | 字段语义转换 | 自动把技术字段名转业务名称（如order_amount→订单金额） | P0 | LLM调用 |
| F01-10 | 表关系识别 | 自动识别表之间的关联关系（一对一/一对多） | P0 | F01-07 |
| F01-11 | 敏感字段识别 | 自动识别手机号/身份证/薪资等敏感字段并标记 | P0 | F01-08 |

### 1.3 数据同步

| 任务ID | 功能点 | 描述 | 优先级 | 技术依赖 |
|--------|--------|------|--------|----------|
| F01-12 | 实时同步模式 | 数据库直连模式，查询时实时取数 | P0 | 数据源连接 |
| F01-13 | 文件同步模式 | Excel/CSV上传后AI自动解析表结构入库 | P0 | 文件解析库 |

## 二、模块二：AI核心分析引擎

### 2.1 自然语言理解（NLU）

| 任务ID | 功能点 | 描述 | 优先级 | 技术依赖 |
|--------|--------|------|--------|----------|
| F02-01 | 意图识别 | 精准识别用户意图（查数/排名/同比环比/异常原因/对比分析） | P0 | LLM调用 |
| F02-02 | 实体抽取 | 自动抽取核心实体（指标/维度/时间/过滤条件） | P0 | F02-01 |
| F02-03 | 信息补全 | 自动补全模糊信息（如"本月业绩"自动补全时间范围） | P0 | F02-02 |
| F02-04 | 追问确认 | 仅当信息缺失/有歧义时极简确认 | P0 | F02-03 |

### 2.2 SQL自动生成

| 任务ID | 功能点 | 描述 | 优先级 | 技术依赖 |
|--------|--------|------|--------|----------|
| F02-05 | SQL生成 | 支持聚合/分组/排序/联表/同比环比等场景 | P0 | F02-02 + 元数据 |
| F02-06 | 多数据源适配 | 适配不同数据源的SQL语法 | P0 | F02-05 |
| F02-07 | SQL语法校验 | 自动校验并修正语法错误（最多3次重试） | P0 | F02-05 |
| F02-08 | SQL性能优化 | 避免全表扫描/慢查询 | P0 | F02-05 |
| F02-09 | 权限校验过滤 | SQL自动叠加用户数据权限过滤条件 | P0 | F04-02 |
| F02-10 | SQL注入防护 | 从底层防止SQL注入 | P0 | F02-05 |
| F02-11 | SQL查看（管理员） | 仅管理员可查看生成的SQL | P1 | - |

### 2.3 数据结果处理

| 任务ID | 功能点 | 描述 | 优先级 | 技术依赖 |
|--------|--------|------|--------|----------|
| F02-12 | 数据格式化 | 统一金额/日期/百分比格式 | P0 | SQL执行结果 |
| F02-13 | 指标计算 | 自动计算同比环比/排名占比 | P0 | F02-12 |
| F02-14 | 关键信息标注 | 异常波动/TOP-Bottom/关键拐点标注 | P0 | F02-13 |
| F02-15 | 业务结论生成 | 生成结构化人话结论 | P0 | F02-14 + LLM调用 |
| F02-16 | 根因分析 | 针对异常波动自动下钻3层维度 | P1 | F02-15 |

### 2.4 多轮对话

| 任务ID | 功能点 | 描述 | 优先级 | 技术依赖 |
|--------|--------|------|--------|----------|
| F02-17 | 上下文记忆 | 自动保留最近10轮对话上下文 | P0 | 会话管理 |
| F02-18 | 条件继承 | 自动继承上一轮的指标/时间/过滤条件 | P0 | F02-17 |
| F02-19 | 层级下钻 | 支持全国→省份→城市→门店层层下钻 | P0 | F02-18 |

## 五、开发优先级排序

### 第一批：MVP核心闭环（P0，优先开发）

这批任务完成后，产品可以跑通核心流程：「用户提问 → AI处理 → 输出答案」

| 模块 | 任务ID | 功能点 |
|------|--------|--------|
| 公共 | F04-01, F04-02 | 角色体系 + 用户创建（先有账号才能用） |
| 数据接入 | F01-01~F01-06 | 数据源管理全流程 |
| 数据接入 | F01-07~F01-11 | AI元数据识别 |
| AI引擎 | F02-01~F02-04 | NLU（意图识别+实体抽取） |
| AI引擎 | F02-05~F02-10 | SQL生成全流程 |
| AI引擎 | F02-12~F02-15 | 结果处理+结论生成 |
| 结果输出 | F03-01~F03-04 | 可视化呈现 |
| 结果输出 | F03-11 | 快捷提问框 |

### 第二批：完善核心体验（P0，补充开发）

| 模块 | 任务ID | 功能点 |
|------|--------|--------|
| AI引擎 | F02-17~F02-19 | 多轮对话 |
| 结果输出 | F03-05~F03-10 | 结果管理+历史查询 |
| 系统管理 | F04-08~F04-10 | 数据权限控制 |
| 系统管理 | F04-11~F04-16 | 操作日志 |

### 第三批：增值功能（P1，可延后）

| 模块 | 任务ID | 功能点 |
|------|--------|--------|
| 数据接入 | F01-12~F01-13 | 数据同步模式 |
| AI引擎 | F02-11 | SQL查看（管理员） |
| AI引擎 | F02-16 | 根因分析 |
| 结果输出 | F03-12~F03-14 | 个人首页完善 |
| 系统管理 | F04-07, F04-17~F04-20 | 批量导入+系统配置 |
